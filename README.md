# Информационная система "Обувь"

## Описание
Разработано для ГИА ДЭ ПУ по специальности 09.02.07.  
Реализует управление товарами, заказами, авторизацию с разными ролями (гость, клиент, менеджер, администратор).

## Технологии
- Python 3.10+
- PyQt5
- SQLite3
- Pandas, Pillow

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать: `venv\Scripts\activate` (Windows) или `source venv/bin/activate` (Linux/Mac)
